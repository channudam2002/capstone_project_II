From upper data vittualization on feature source_system_tab, source_screen_name and ***source_type** we can assume that:

  • source_system_tab
    • user that listen repetitively mostly they browsed from my library as top 01, discover as top 02 and search as top 03
    • user that listen only onece mostly they browsed from my library as top 01, discover as top 02 and radio as top 03
  • source_screen_name
    • user that listen repetitively mostly they browsed from Local playlist more as top 01, Online playlist more as top 02 and Album more as top 03
    • user that listen only onece mostly they browsed from Local playlist more as top 01, Online playlist more as top 02 and Radio as top 03
  • source_type
    • user that listen repetitively mostly they browsed from local-library as top 01, online-playlist as top 02 and local-playlist as top 03
    • user that listen only onece mostly they browsed from online-playlist as top 01, local-library as top 02 and radio as top 03

So

  • we can keep only my library, discover, search and radio for column source_system_tab
  • we can keep only Local playlist more, Online playlist more, Album more, Radio for column source_screen_name
  • we can keep only local-library, online-playlist, local-playlist, radio for column source_type

And also

  • Replace null by the top category based on target column
